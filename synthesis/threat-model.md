# Threat Model

## Assets

- Customer Data
- Financial Records
- Active Directory
- Internal Workstations
- Web Applications

## Threat Actors

- Cybercriminals
- Malware Operators
- External Attackers

## Attack Vectors

- Malicious File Download
- Phishing
- SQL Injection
- Cross-Site Scripting
- Weak Authentication

## Identified Risks

1. Malware infection through suspicious download
2. Unauthorized database access via SQLi
3. Session hijacking via XSS
4. Credential compromise
5. Data exfiltration

## Business Impact

- Data breach
- Financial loss
- Regulatory penalties
- Reputational damage
